    unity插件使用教程

    一.unity软件使用

    1.打开Unity Hub软件(若未下载可官网下载：https://unity.cn/releases)

    {width="7.791666666666667in" height="5.697656386701662in"}

    2.检查是否安装编辑器（最低可用版本2019.1.0）

    {width="7.791666666666667in" height="4.159613954505687in"}

    若未下载可点击安装unit编辑器按钮，按照unity编辑器或者官网下载（地址：https://unity.cn/releases/full/2019）

    {width="7.791666666666667in" height="4.119032152230971in"}

    3.新建项目或打开已有项目

    {width="7.791666666666667in" height="3.8998917322834648in"}

    新建项目

    {width="7.791666666666667in" height="4.210504155730534in"}

    创建项目后自动打开新建的项目

    {width="7.791666666666667in" height="4.157994313210849in"}

    4.打开后如果存在警告(无警告可跳过此步骤)

    {width="7.791666666666667in" height="4.2776979440069995in"}

    点击window-package Manager

    {width="7.791666666666667in" height="4.2004079177602796in"}

    点击Visual Studio Editor--Update to 2.0.22

    {width="7.791666666666667in" height="4.190203412073491in"}

    更新完成后关闭弹窗

    {width="7.791666666666667in" height="4.24334864391951in"}

    5.导入unity插件

    unity插件2.3.0版本下载

    请至钉钉文档查看附件《mostechPlugin2.3.0.unitypackage》

    将motionPlugin2.1.0插件文件拖拽至Project-Assets里面

    {width="7.791666666666667in" height="4.509662073490814in"}

    点击import按钮

    {width="7.791666666666667in" height="4.143897637795275in"}

    6.驱动全身设备

    点击assets---mostech---scenes

    {width="7.791666666666667in" height="4.183055555555556in"}

    点击Motion Animator Scene或Motion Transforms Scene

    {width="7.791666666666667in" height="4.222010061242345in"} 在Hierarchy点击MotionDriver

    {width="7.791666666666667in" height="4.226195319335083in"}

    点击inspector---Motion Source Manager(script)输入motion中对应的ip和端口

    {width="7.791666666666667in" height="4.224176509186352in"}

    连接设备后，并做好姿势校准

    {width="7.791666666666667in" height="4.202075678040245in"}

    点击运行，可传输实时数据

    {width="7.791666666666667in" height="4.27265091863517in"}

    7.驱动手套设备运行

    点击assets---mostech---scenes

    {width="7.791666666666667in" height="4.193867016622922in"}

    点击Motion Hand Scene

    {width="7.791666666666667in" height="4.219214785651793in"}

    点击inspector---Motion Source Manager(script)输入motion中对应的ip和端口

    {width="7.791666666666667in" height="4.214514435695538in"}

    连接设备后，并做好姿势校准

    {width="7.791666666666667in" height="4.23420384951881in"}

    点击运行，可传输实时数据

    {width="7.791666666666667in" height="4.279886264216973in"}

    8.驱动组合设备运行

    选择的模特与全身设备一样

    {width="7.791666666666667in" height="4.209150262467191in"}

    点击inspector---Motion Source Manager(script)输入motion中对应的ip和端口

    {width="7.791666666666667in" height="4.21336176727909in"}

    motion上将设备组合起来，并做好姿势校准

    {width="7.791666666666667in" height="4.220613517060367in"}

    点击运行，可传输实时数据

    {width="7.791666666666667in" height="4.191758530183727in"}

    9.驱动多设备运行

    在Hierarchy选中Mostech-Robot进行粘贴复制

    {width="7.791666666666667in" height="4.2447364391951in"}

    点击inspector---Motion Instances输入对应的模特数

    {width="7.791666666666667in" height="4.259972659667541in"}

    点击motionDriver再点击inspector---Motion Instances---Element0、Element1选择相对应的Mostech-Robot

    {width="7.791666666666667in" height="4.12749343832021in"}

    点击Mostech-Robot再点击inspector---Actor ID输入不同的数字

    {width="7.791666666666667in" height="4.258912948381452in"}

    点击Mostech-Robot再点击inspector---模特位置调整，修改x,y,z

    {width="7.791666666666667in" height="4.23837489063867in"}

    {width="7.791666666666667in" height="4.224800962379702in"}

    motion上做好姿势校准后，untiy上点击运行按钮

    {width="7.791666666666667in" height="4.382812773403325in"}

    10.一套设备驱动多个模特

    在Hierarchy选中Mostech-Robot进行粘贴复制

    {width="7.791666666666667in" height="4.2447364391951in"}

    点击inspector---Motion Instances输入对应的模特数

    {width="7.791666666666667in" height="4.259972659667541in"}

    点击motionDriver再点击inspector---Motion Instances---Element0、Element1选择相对应的Mostech-Robot

    {width="7.791666666666667in" height="4.12749343832021in"}点击Mostech-Robot再点击inspector---Actor ID输入相同的数字

    {width="7.791666666666667in" height="4.258912948381452in"}

    点击Mostech-Robot再点击inspector---模特位置调整，修改x,y,z

    {width="7.791666666666667in" height="4.236719160104987in"}

    {width="7.791666666666667in" height="4.224800962379702in"}

    motion上做好姿势校准后，untiy上点击运行按钮

    {width="7.791666666666667in" height="4.20279636920385in"}

    二.motion studio软件数据广播使用

    1.打开Motion Studio软件

    {width="5.041666666666667in" height="5.635416666666667in"}

    2.进入主页界面，可新建项目或者打开已新建好的项目

    {width="7.791666666666667in" height="4.053491907261592in"}

    {width="7.791666666666667in" height="4.094137139107612in"}

    3.选中项目后，进入捕捉页面

    {width="7.791666666666667in" height="4.207744969378828in"}

    4.开启数据广播

    点击左上角功能-数据广播按钮

    {width="7.791666666666667in" height="4.208311461067367in"}

    开启bvh广播（端口号可随意输入）

    {width="7.791666666666667in" height="4.198639545056868in"}

    {width="7.791666666666667in" height="4.198639545056868in"}

    5.数据广播开启后，未连接设备可播放离线数据

    查看文件列表，双击选择数据文件播放

    {width="7.791666666666667in" height="4.200512904636921in"}

    播放文件

    {width="7.791666666666667in" height="4.173523622047244in"}

    unity IP和端口配置完成后，点击播放按钮，可播放离线数据

    {width="7.791666666666667in" height="4.188937007874015in"}

    6.数据广播开启后，连接设备可播放实时数据

    接收器连接电脑，当设备列表有设备显示时，点击连接按钮进行设备连接

    {width="7.791666666666667in" height="4.239583333333333in"}

    连接成功后，可将绑带和设备按照提示依次穿戴在身上

    {width="3.3229166666666665in" height="4.322916666666667in"}

    穿戴完成，点击姿势校准，根据提示进行T-Pose、A-Pose、OK-Pose(有手套设备组合时)姿势校准

    {width="7.791666666666667in" height="4.2314063867016625in"}

    {width="7.791666666666667in" height="4.2217672790901135in"}

    {width="7.791666666666667in" height="4.240320428696413in"}

    unity IP和端口配置完成后，点击播放按钮，可播放实时数据

    {width="7.791666666666667in" height="4.2230260279965in"}

    {width="7.791666666666667in" height="4.186383420822398in"}
